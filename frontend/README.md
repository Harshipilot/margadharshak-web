# Margadarshak Frontend

A React + Tailwind frontend for the Margadarshak rural education platform.

## Setup

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Features

- Landing page with student/adult/offline assistant entry points
- Age-based dashboard and profile experience
- Sahayak offline AI calling page
- Modular Tailwind-based UI components
- API client with JWT auth support

## Notes

- Backend API proxy is configured in `vite.config.js`
- Ensure backend is running at `http://localhost:5000`
- Add additional pages in `src/pages` for expanded modules
